# 🎓 Student Management System (Python)

A simple command-line **Student Management System** built in Python to practice Object-Oriented Programming (OOP), CRUD operations, file handling, and JSON data persistence.

This project is the foundation of my backend learning journey. Future versions will replace JSON with SQL databases and later expose the functionality through a FastAPI REST API.

---

## 🚀 Features

* Add Student
* View All Students
* Search Student by SID
* Update Student Details
* Delete Student
* Automatic JSON Saving
* Automatic JSON Loading
* Duplicate SID Validation
* Integer Input Validation
* Exception Handling for Missing Database File

## ▶️ How to Run

1. Clone the repository.

```
git clone <repository-url>
```

2. Navigate to the project folder.

```
cd StudentManagement
```

3. Run the application.

```
python student_manager.py
```
